Hi, my name is Jeff and I have some advice for you!

Practice makes perfect!

Writing code may not be intuitive at first, but neither is riding a bike! With persistence and consistency, writing code to create will become second nature. The more code we write, the faster and more fluid it will become.

Create something for you.

While these are technical skills that we're learning, they are powerful creative tools as well. Building a website or portfolio that allows you to express yourself is a great way to feel connected to the skills you're learning, and serves as a great motivator to keep pushing forward!
